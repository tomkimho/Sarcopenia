## Quick Start (5분)

```powershell
# 1. 환경 설정
cd sarcopenia_dcp\scripts
pip install openpyxl anthropic requests pandas streamlit plotly pdfplumber

# 2. API 키 설정
$env:ANTHROPIC_API_KEY = "sk-ant-api03-..."

# 3. 환경 테스트
python 00_env_test.py

# 4. 웹 대시보드 실행 (데이터 있는 경우)
streamlit run 09_website.py
```

## 전체 파이프라인 실행

### 방법 1: 한 번에 실행 (오케스트레이터)

```powershell
# 일일 업데이트 (수집 + 추출 + 분석)
python scripts/08_orchestrator.py

# 주간 전체 분석 (Dark Target + 신약 후보 포함)
python scripts/08_orchestrator.py --weekly
```

### 방법 2: 단계별 실행

```powershell
# Step 1: PDF 텍스트 추출 (~30분)
python scripts/01_pdf_extract.py

# Step 2: Claude AI 정보 추출 (~8-16시간, 밤에 돌려놓기)
python scripts/02_info_extract.py

# Step 3: 화합물 구조 수집 (~30분, 무료)
python scripts/03_compound_structure.py

# Step 4: 천연물 활성성분 매핑 (~10분, 무료)
python scripts/04_natural_products.py

# Step 5: PubMed 신규 논문 수집 (~30분)
python scripts/05_pubmed_collect.py --years 2

# Step 6: 특허 수집 (~5분)
python scripts/06_patent_collect.py

# Step 7: bioRxiv 프리프린트 수집 (~5분)
python scripts/07_biorxiv_collect.py

# Step 8: 바이오마커 분석 (~5분, 무료)
python scripts/12_biomarker_analysis.py

# Step 9: Dark Target 발굴 (~5분, 무료)
python scripts/10_pattern_analysis.py

# Step 10: AI 신약 후보 도출 (~5분, ~$0.5)
python scripts/11_drug_candidates.py

# Step 11: 웹 대시보드 실행
streamlit run scripts/09_website.py
```

## PDF 파일 배치

현재 3,894개 PDF가 상위 폴더에 있습니다.
01_pdf_extract.py는 자동으로 상위 폴더를 탐색합니다.

원하시면 분류 가능:
```
sarcopenia_dcp/
├── totalpaper/
│   ├── papers/    ← 논문 PDF
│   └── patents/   ← 특허 PDF
```

## 주요 파라미터

| 파라미터 | 설정 | 효과 |
|---------|------|------|
| `--append` (02) | APPEND 모드 | 기존 데이터 유지 + 신규만 추가 |
| `--all` (05) | 30년 전체 | 대량 수집 |
| `--years N` (05) | 최근 N년 | 빠른 수집 |
| `--weekly` (08) | 주간 분석 | Dark Target + 신약 후보 포함 |
